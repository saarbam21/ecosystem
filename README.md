# Ecosystem — אתר תכנון פיננסי

אתר תדמית בעברית (RTL) שנבנה מחדש ב-**Next.js** (App Router) עם **Tailwind CSS**, מיועד לאירוח **חינמי בוורסל (Vercel)** כאתר סטטי.

## מבנה

- `app/` — העמודים (בית, שירותים, בלוג, שאלות נפוצות, אודות, צור קשר, פרטיות)
- `components/` — רכיבים משותפים (Header, Footer, טופס, CTA וכו')
- `content/blog/` — פוסטים של הבלוג בקבצי Markdown
- `lib/site.js` — **כל פרטי הקשר והקבועים במקום אחד** (טלפון, וואטסאפ, מייל, מפתח טופס)
- `lib/posts.js` — קריאת פוסטי הבלוג

## הרצה מקומית

```bash
npm install
npm run dev        # http://localhost:3000
npm run build      # יוצר אתר סטטי בתיקיית out/
```

## הוספת פוסט חדש לבלוג

יוצרים קובץ חדש ב-`content/blog/`, למשל `my-post.md`:

```markdown
---
title: "כותרת הפוסט"
date: "2026-06-19"
author: "סער בם"
excerpt: "תקציר קצר שמופיע בעמוד הבלוג."
---

תוכן הפוסט כאן, ב-Markdown רגיל...
```

הכתובת תהיה אוטומטית `/blog/my-post/`.

## העלאה לוורסל (חינם)

### דרך GitHub (מומלץ)
1. מעלים את התיקייה לריפו ב-GitHub.
2. נכנסים ל-https://vercel.com , מתחברים עם GitHub, ולוחצים **Add New → Project**.
3. בוחרים את הריפו. Vercel מזהה Next.js אוטומטית — לוחצים **Deploy**.
4. תוך דקה האתר עולה בכתובת `xxx.vercel.app`.

### חיבור הדומיין ecosystemplan.com
ב-Vercel: **Project → Settings → Domains → Add**, מזינים `ecosystemplan.com`, ומעדכנים אצל ספק הדומיין את רשומות ה-DNS לפי ההוראות שוורסל מציג.

> שימו לב: כשמעבירים את הדומיין לוורסל, האתר הישן בוורדפרס יפסיק להופיע בכתובת הזו. כדאי לוודא שהכל תקין באתר החדש קודם.
